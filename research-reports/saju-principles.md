# 사주 풀이의 원리 — 계산이 어떻게 나오는지

이 문서는 명리학 표준 교과서(자평진전·적천수·궁통보감)에 적힌 풀이 절차를 정리한 것이다. **thegot이 실제로 어떤 알고리즘/가중치를 쓰는지는 코드를 안 깠으니 모른다** — thegot 출력 카드를 보면 이 단계들의 산물(강약 점수, 격국, 용신, 관인상생/상관견관 라벨, 신살 등)이 다 표시되어 있어서 어느 단계든 거치긴 했다는 건 확인됨. 하지만 점수 가중치, 학파 우선순위, 신살 데이터셋 등 세부는 불명. 다른 사이트(포스텔러 등)는 직접 까보지 않았다.

---

## 0. 사주의 입력과 1차 출력

**입력:** 생년월일시 + 성별 + 양/음력 토글 + 진태양시·서머타임 보정.

**1차 변환 — 만세력 + 절기 보정으로 4기둥 만들기:**

생년월일시 → (절기 기준으로 월·년 경계 보정) → 4기둥 8글자.
- 년주: 입춘이 새해 경계. 양력 1월생이라도 입춘(2월 4일경) 전이면 전년 간지.
- 월주: 12절(節)이 경계. 매월 절입일 전이면 전월 간지.
- 일주: 60갑자 순환. 1900년대 만세력 한 번 박아놓으면 일자 차이 mod 60.
- 시주: 일간 + 출생 시각 → 시간지지 12개 중 1 → 시간천간은 일간×시지 공식 (2시간 단위).

여기까지는 그냥 **달력 변환**. 명리학적 풀이는 1차 출력을 다 받아놓고 시작한다.

## 1. 지장간 — 모든 풀이의 기반

지지(地支)는 단순한 12동물이 아니다. **각 지지 안에 천간 2~3개가 숨어있다 (지장간/장간).** 이게 사주 풀이의 가장 중요한 1차 데이터.

각 지지마다:
- **여기(餘氣)** — 앞 절기의 잔여 기운. 절입 직후엔 강하고 시간이 지날수록 약해짐.
- **중기(中氣)** — 삼합의 중심 기운. 일부 지지에만 있음.
- **본기/정기(本氣/正氣)** — 그 지지의 대표 천간. 가장 강하고 모두에게 있음.

예시:
- 인(寅): 무(7일) → 병(7일) → **갑**(16일) — 본기는 갑목
- 자(子): 임(10일) → **계**(20일) — 본기는 계수
- 술(戌): 신(9일) → 정(3일) → **무**(18일) — 본기는 무토 (화 묘지)

**왜 중요한가:**

(a) **격국**은 월지의 지장간 중 어느 것이 천간으로 투출됐는지로 정한다 (§5).

(b) **통근(通根)** — 천간이 지지 안에 자기 오행을 가지고 있으면 "뿌리가 있다"고 한다. 갑목 천간이 인지나 묘지 위에 있으면 강하게 통근. 술지 위에 있으면 통근 안 됨. 통근 여부가 그 천간의 힘을 결정한다.

(c) **투간(透干)** — 반대 방향. 지장간이 천간으로 올라와 있으면 "투간했다"고 한다. 월지 지장간이 투간하면 그게 격국이 된다 (대개).

지장간 표는 명리학 표준이라 어느 책이든 같다. 60갑자 일주 + 12지지 모두에 대해 표 한 번 외우면 나머지 풀이는 다 이 위에 얹힌다.

[참고: 사주명리 입문 - 지장간](https://www.sajustudy.com/31)

## 2. 십성(十神) — 일간 기준의 관계 라벨링

일간은 사주의 주인공("나"). 다른 7글자 (년간/월간/시간 + 4지지의 본기) 각각이 일간과 어떤 오행 관계를 맺는지로 **10개의 라벨** 중 하나를 받는다.

판정 규칙:
1. 두 천간의 오행 관계 본다 (생/극/같음).
2. 두 천간의 음양이 같으면 한 종류, 다르면 다른 종류.

| 오행 관계 | 음양 같음 | 음양 다름 |
|---|---|---|
| 같은 오행 | 비견 (比肩) | 겁재 (劫財) |
| 내가 생함 (자식) | 식신 (食神) | 상관 (傷官) |
| 내가 극함 (재물) | 편재 (偏財) | 정재 (正財) |
| 나를 극함 (관/책임) | 편관 (偏官, 칠살) | 정관 (正官) |
| 나를 생함 (어머니) | 편인 (偏印) | 정인 (正印) |

5쌍을 카테고리로 묶으면: **비겁 / 식상 / 재성 / 관성 / 인성**.

이 5카테고리 카운트가 사주 분석의 모든 후속 단계에 들어간다. 강약 점수도, 격국도, 용신도, 십성 조합 패턴 라벨도 다 이 카운트에서 나옴.

[참고: 십성 위키백과](https://ko.wikipedia.org/wiki/%EC%8B%AD%EC%84%B1_(%EC%82%AC%EC%A3%BC%ED%8C%94%EC%9E%90))

## 3. 강약 판정 — 일간이 얼마나 힘이 있는가

**왜 강약이 핵심인가:** 명리학에서 사주 풀이의 첫 분기점. 일간이 강한지 약한지에 따라 용신이 정반대로 정해지고, 모든 권고가 뒤집힌다.

### 3.1 4지표 (득령/득지/득시/득세)

월령(月令)이 압도적으로 중요. 일지·시지가 그 다음. 년지는 약함. 4가지 boolean으로 판정:

**득령(得令) — 월지가 일간을 돕는가?**
월지 본기가 (a) 일간과 같은 오행 = 비겁이거나 (b) 일간을 생하는 오행 = 인성이면 득령.
명리학에선 월령이 일간 힘의 50%를 차지한다고 본다. 득령 1번이 다른 지표 2-3번 가치.

**득지(得地) — 일지가 일간을 돕는가?**
일지 본기가 비겁이거나 인성이면 득지. 일지는 부부궁이자 일간이 깔고 앉은 자리라 25% 정도의 가중치.

**득시(得時) — 시지가 일간을 돕는가?**
시지가 비겁이나 인성이면 득시. 가중치는 일지보다 약간 낮음.

**득세(得勢) — 사주 전체에 비겁/인성이 많은가?**
일간 제외하고 사주 8글자 (4천간 + 4지지 본기) 중 비겁 + 인성이 3개 이상이면 득세.

### 3.2 점수화

전통 방식은 4지표 boolean만 쓰지만, 현대 명리 사이트는 0~100 점수화를 쓴다. 표준화된 공식은 없지만 핵심 로직은:

```
점수 = 0
+ 월령 가중치 × (월지가 비겁/인성/식상/재성/관성 중 어느쪽이냐)
  - 비겁: +30, 인성: +20, 식상: -10, 재성: -15, 관성: -15
+ 일지 가중치 × ... (월령의 절반 정도)
+ 시지 가중치 × ... 
+ 천간 4개에 대해 각각 비겁=+5, 인성=+3, 그 외=음수
+ 지장간 본기 + 중기 + 여기 가중치 (본기 100%, 중기 30%, 여기 10%)
+ 월령에서 절입 후 며칠 지났는지로 본기/중기/여기 비율 조정
= 0~100 normalize
```

판정 등급:
- 50 미만: 신약 (일간이 약함, 도움이 필요)
- 50~70: 중화신강 (적당히 강함, 거의 균형)
- 70~85: 신강 (꽤 강함, 출구 필요)
- 85+: 태강/극왕 (너무 강함, 종강격 가능성)
- 15 미만: 극약 (너무 약함, 종약격 가능성)

극단 구간은 일반 룰이 뒤집히는 변격으로 간다 (§5.2).

### 3.3 여기서 분기되는 것들

강약 판정 한 번이 결정하는 것:
- 용신 방향 (강하면 설기, 약하면 보강 — §4)
- 길운/흉운 판정 (대운 십성이 일간을 더 강하게 만드냐 약하게 만드냐로 평가 뒤집힘)
- 격국이 정격이냐 변격이냐
- "이 사주에 좋은 색·방향·음식·직업"의 모든 출처

## 4. 용신(用神) — 사주에 가장 필요한 글자

용신은 "사주의 균형을 맞추기 위해 필요한 오행." 한 사주에 하나만 있다. 이게 정해지면 풀이의 거의 모든 권고가 자동으로 결정됨 — 좋은 색, 좋은 방향, 좋은 음식, 좋은 직업, 좋은 대운, 좋은 달.

문제는 **용신 정하는 방식이 명리학파마다 다름**. 그래서 thegot처럼 진지한 사이트는 3종을 다 표시한다.

### 4.1 억부용신(抑扶用神) — 가장 흔함, 강약 기반

> **강한 것은 억제, 약한 것은 부조**

알고리즘:
1. 강약 판정 (§3).
2. 신강 → 일간을 빼는 오행이 용신:
   - 식상 (자식 — 일간 에너지를 발산)
   - 재성 (내가 극하는 — 에너지 배출)
   - 관성 (나를 극하는 — 압박해서 빼냄)
   세 후보 중 **사주에 이미 있고 힘 있는** 것을 우선.
3. 신약 → 일간을 돕는 오행이 용신:
   - 인성 (생함)
   - 비겁 (같은 편)
   둘 중 더 적당한 쪽.
4. 중화 → 미세 조정. 어느 한쪽으로 약간 기울었으면 그 반대를 보강.

전체 사주의 약 70%가 이 방식으로 용신 결정.

### 4.2 조후용신(調候用神) — 한난조습 기반

> **너무 추우면 따뜻하게, 너무 더우면 시원하게**

알고리즘:
1. 월지를 본다 (계절).
2. 일간과 월지 조합으로 한난조습 판정:
   - 한기: 자/축/해 월에 태어났고 화 부족 → 화가 용신
   - 열기: 사/오/미 월에 태어났고 수 부족 → 수가 용신
   - 조기(마름): 미/술 월에 태어났고 수 부족 → 수가 용신
   - 습기(축축함): 진/축 월에 태어났고 화 부족 → 화가 용신

전통 만세력 책 『궁통보감』에 일간 × 월지 100칸 표가 있고, 이게 표준.

### 4.3 격국용신(格局用神) — 격에 맞는 흐름 기반

> **격국이 정해지면, 그 격이 살아나는 방향이 용신**

알고리즘:
1. 격국 결정 (§5).
2. 격이 좋은 격(정관격/정인격/식신격/정재격) → 그 격을 보호하는 오행이 용신.
3. 격이 나쁜 격(편관격/상관격/편인격/겁재격) → 그 격을 제어/순화하는 오행이 용신.

예: 정관격 → 정관(관성)을 인성이 받아주거나(관인상생) 재성이 도와줌(재생관). 그래서 인성 또는 재성이 용신.

### 4.4 우선순위 — 학파마다 다름

전통 명리학의 충돌 해소 룰:
1. **별격(종격/화격/양기성상격) 먼저 본다.** 사주가 한 오행에 극단적으로 치우쳤으면 그 흐름을 따르는 게 용신. 이건 다른 두 학파를 무시.
2. **조후가 다급하면 조후 우선.** 한겨울 병화나 한여름 임수처럼 계절 충격이 심하면 일단 조후 살리고 본다.
3. **그 외엔 억부.** 70% 사주가 여기.
4. 격국용신은 정통파가 1순위로 쓰지만 현대 사이트는 보조.

용신이 결정되면:
- **희신(喜神):** 용신을 도와주는 오행
- **한신(閑神):** 중립
- **기신(忌神):** 용신을 해치는 오행
- **구신(仇神):** 기신을 도와주는 오행

이 5단계 라벨링이 모든 대운/세운 길흉 판정의 기준이 된다.

[참고: 용신 위키백과](https://ko.wikipedia.org/wiki/%EC%9A%A9%EC%8B%A0_(%EC%82%AC%EC%A3%BC%ED%8C%94%EC%9E%90)), [정해만세력 용신 강의](https://doc.8-codes.com/docs/lecture/15/)

## 5. 격국(格局) — 사주의 구조적 패턴

격국은 사주를 50만 개 → 한 자릿수 카테고리로 묶는 분류 체계. **월지가 가장 강한 자리**라는 명리학 공리에 기반.

### 5.1 정격 (내격) 결정 알고리즘

표준 절차:

```
1. 월지의 지장간 (본기/중기/여기) 본다.
2. 본기가 천간 4개 중 어디에 투간됐는지 확인.
   → 투간했으면 그 본기의 십성이 격국. (예: 월지 인의 본기 갑이 천간에 있으면 비견격 또는 겁재격)
3. 본기가 투간 안 됐으면 중기 확인.
   → 중기가 천간에 있으면 중기의 십성이 격국.
4. 중기도 투간 안 됐으면 여기 확인.
5. 셋 다 투간 안 됐으면 → 그냥 월지 본기의 십성을 격으로 채택 (잠재격국).
```

→ 비견격 / 겁재격 / 식신격 / 상관격 / 편재격 / 정재격 / 편관격 / 정관격 / 편인격 / 정인격 — 10정격.

월지가 자기 일간이면(예: 갑목 일간이 인월) → **건록격**, 월지가 일간의 제왕지면(예: 갑목이 묘월) → **양인격**. 이 둘은 사실상 비견격/겁재격이지만 별도 라벨.

→ 총 12정격 (우리 `gyeokguk.json`에 있는 12개).

### 5.2 변격(외격)

사주가 한 오행에 극단적으로 치우치면 정격 룰이 깨지고 그 흐름을 그대로 따라가야 한다.

- **종강격(從强格):** 비겁이 5개 이상, 일간이 너무 강함 → 비겁 흐름을 그대로 인정
- **종왕격(從旺格):** 비겁 + 인성이 압도적
- **종재격(從財格):** 재성이 5개 이상, 일간 약하면 재성에 종속
- **종관살격(從官煞格):** 관성이 압도적
- **종아격(從兒格):** 식상이 압도적
- **화격(化格):** 일간이 다른 천간과 합화하여 다른 오행으로 변함 (예: 갑기합토)
- **양기성상격(兩氣成象格):** 사주가 두 오행으로만 구성

변격일 땐 억부용신을 안 쓰고 그 흐름의 주축 오행이 곧 용신.

## 6. 십성 조합 패턴 — 격국 위에 올라가는 두 번째 라벨링

격국 하나만으론 부족하다. 사주 8글자가 서로 어떤 **흐름**을 만드는지가 풀이의 깊이를 결정.

### 6.1 핵심 원리: 5카테고리의 상생 vs 상극

5카테고리 (비겁·식상·재성·관성·인성)가 5각형 위에 배치되어 있고:
- 상생 (시계방향): 비겁 → 식상 → 재성 → 관성 → 인성 → 비겁
- 상극 (별 모양): 비겁이 재성을 극, 식상이 관성을 극, 재성이 인성을 극, 관성이 비겁을 극, 인성이 식상을 극

이 위에서 사주에 어떤 카테고리가 인접 배치돼 있는지로 패턴 라벨이 붙는다.

### 6.2 주요 길격 패턴

**관인상생(官印相生) — 관성이 인성을 통해 일간으로 흐름**
- 룰: 사주에 관성과 인성이 둘 다 있고, 위치가 관성 → 인성 → 일간 순으로 생을 이어줌.
- 의미: 관성(나를 극하는 압박, 직장/조직)이 직접 일간을 치지 않고 인성(엄마, 학문)을 거쳐 일간을 도와주는 형태. 압박이 곧 자양분.
- 적성: 공무원, 교수, 대기업 — 조직에서 인정받으며 승진.

**살인상생(殺印相生) — 편관(칠살) 버전**
- 관인상생인데 관성이 편관(칠살)인 경우. 시련이 더 격하지만 인성이 받아주면 더 큰 인물.

**식상생재(食傷生財) — 재능이 돈으로**
- 룰: 식상이 재성을 생하는 위치. 일간 → 식상 → 재성.
- 의미: 자기 표현/재능/창의가 곧 수익. 콘텐츠, 강의, 전문직, 자영업.

**재생관(財生官) — 돈이 직위로**
- 룰: 재성이 관성을 생하는 위치.
- 의미: 부와 권력이 같이 옴. 재력가가 정치/사회적 지위까지.

**재자약살(財滋弱殺) / 살인상정(殺刃相停) 등 30+개의 고전적 명리학 패턴이 있음.**

### 6.3 주요 흉격 패턴

**상관견관(傷官見官) — 상관이 정관을 침**
- 룰: 사주에 상관과 정관이 같이 있고, 상관이 정관을 극하는 위치.
- 의미: 자유로운 기질(상관)이 규범(정관)과 충돌. 직장에서 상사와 마찰, 조직 부적응.
- 완화책: 인성이 둘 사이를 통관해주면 누그러짐.

**비겁쟁재(比劫爭財) — 형제가 재물을 다툼**
- 룰: 비겁이 많은데 재성이 적음.
- 의미: 경쟁자/동료/형제가 내 재물을 가져가려 함. 동업 불리.
- 완화책: 관성이 비겁을 제어하거나 식상으로 에너지를 발산.

**탐재괴인(貪財壞印) — 재물을 탐해 인성을 깸**
- 룰: 재성이 인성을 극하는 위치 + 재성이 강함.
- 의미: 돈에 눈멀어 학문/도덕/명예를 잃음.

**식상제살(食傷制殺) 미완 / 관살혼잡(官殺混雜) 등.**

이런 패턴 라벨이 사주 풀이에서 **굵게 강조되는 4글자 한자어**들. "넘치는 힘에 출구가 없는 사주"라는 한 줄 정의는 이 패턴 분석의 자연어 표현.

[참고: 관인상생, 살인상생의 현대적 고찰](https://brunch.co.kr/@@2ak2/397), [재생관/재생상, 관인상생/살인상생](https://gall.dcinside.com/board/view/?id=divination_new1&no=5780414)

## 7. 합·충·형·파·해 — 글자 사이의 관계 작용

격국과 십성 패턴이 "사주의 평균적 구조"라면, 합충형파해는 "글자 두 개의 관계 이벤트". 사주 안에 어떤 글자 쌍이 있느냐로 자동 트리거.

### 7.1 합(合) — 끌어당김

**천간합 5쌍** — 두 천간이 만나 새 오행으로 변하려 함:
| 합 | 변하는 오행 | 별명 |
|---|---|---|
| 갑 + 기 | 토 | 중정지합 (정직) |
| 을 + 경 | 금 | 인의지합 (예술+실행) |
| 병 + 신 | 수 | 위제지합 (지혜) |
| 정 + 임 | 목 | 인수지합 (감정) |
| 무 + 계 | 화 | 무정지합 (겉치레) |

**지지육합 6쌍** — 가장 친밀:
자축(토) / 인해(목) / 묘술(화) / 진유(금) / 사신(수) / 오미(화토)

**삼합 4그룹** — 세 지지가 모이면 강한 局 형성:
인오술(화국) / 사유축(금국) / 신자진(수국) / 해묘미(목국)
세 글자 모이면 강력. 두 글자만 있으면 반합 — 약함.

**방합 4그룹** — 같은 계절 세 지지:
인묘진(봄/목) / 사오미(여름/화) / 신유술(가을/금) / 해자축(겨울/수)
방합은 더 순수하지만 좀 더 좁음.

### 7.2 충(沖) — 부딪힘

**지지육충 6쌍** — 정반대 방향, 변화·이동·갈등:
자오(수↔화) / 축미(토↔토) / 인신(목↔금) / 묘유(목↔금) / 진술(토↔토) / 사해(화↔수)

월충은 직장/사회 충돌, 일충은 부부 갈등, 시충은 자녀 문제. 충은 무조건 나쁜 게 아니다 — 정체된 기운을 깨뜨려 변화 만듦.

### 7.3 형(刑) — 시련

**3가지 성격:**
- 인사신 = **무은지형(無恩之刑)** — 은혜를 원수로
- 축술미 = **지세지형(恃勢之刑)** — 권력을 믿고 강압
- 자묘 = **무례지형(無禮之刑)** — 도덕적 일탈
- 진진/오오/유유/해해 = **자형(自刑)** — 스스로를 해침

형은 충보다 은밀하게 작용. 법적 분쟁, 건강 이상, 관계 파탄.

### 7.4 파(破)·해(害)

파는 합 안에 든 작은 균열. 해는 합을 방해하는 외부 작용. 둘 다 충/형보다 약함.

## 8. 신살(神殺) — 특수 트리거 룰

### 8.1 두 가지 트리거 방식

**(a) 일간 기준 신살** — 일간이 어떤 천간이냐에 따라 특정 지지가 신살.
- 천을귀인: 일간 갑/무 → 사주에 축이나 미 있으면 트리거
- 양인살: 일간이 양간(갑·병·무·경·임)이고 사주에 그 일간의 제왕지(겁재 지지)가 있으면. 갑→묘, 병/무→오, 경→유, 임→자.
- 문창귀인: 일간 → 식신 지지. 갑→사, 을→오, 병→신, 정→유, ...

**(b) 일지/년지 삼합 기준 신살** — 일지(또는 년지)가 어느 삼합국에 속하느냐로:
- 도화살 = 삼합의 **목욕지(왕지)**:
  - 인오술 삼합 → 묘
  - 사유축 삼합 → 오
  - 신자진 삼합 → 유
  - 해묘미 삼합 → 자
  - 즉 일지가 인/오/술 중 하나이고 사주 어딘가에 묘가 있으면 도화살 트리거
- 역마살 = 삼합의 **장생지(생지)**:
  - 인오술 → 신, 사유축 → 해, 신자진 → 인, 해묘미 → 사
  - 일지/년지 삼합 그룹 + 그 그룹의 충지가 사주에 있으면
- 화개살 = 삼합의 **묘지(고지)**:
  - 인오술 → 술, 사유축 → 축, 신자진 → 진, 해묘미 → 미

### 8.2 왜 이렇게 정해지나 — 삼합의 자리

삼합이 인오술이면:
- 인 = 화의 시작(생지) → 역마살 (변화의 자리)
- 오 = 화의 절정(왕지) → 도화살 (매력의 자리)
- 술 = 화의 끝(묘지) → 화개살 (저장·정신의 자리)

즉 신살은 **삼합 한 묶음에서 어느 자리냐**로 정해진다. 신살 100가지가 전부 이런 식의 자리 매핑이다.

### 8.3 사주에 동시 트리거되는 신살 수

평범한 사주가 보통 5~10개. 많으면 14개도 트리거. thegot이 14개 잡은 건 카운트 인플레가 아니라 **데이터 풀이 풍부**해서. 우리 `sinsal.json`은 8개만 있음.

[참고: 신살 위키백과](https://ko.wikipedia.org/wiki/%EC%8B%A0%EC%82%B4), [12신살 이야기](https://chunmyung.com/community/posts/97)

## 9. 12운성 — 일간의 라이프사이클

12운성(十二運星)은 **일간 천간이 각 지지에서 어느 단계의 에너지냐**를 보는 시스템.

### 9.1 12단계

장생(시작·1) → 목욕(2) → 관대(3) → 건록(4) → **제왕(5, 정점)** → 쇠(6) → 병(7) → 사(8) → 묘(9) → 절(10, 바닥) → 태(11) → 양(12) → 다시 장생.

각 천간마다 12지지에서 자기가 어디에 있느냐 표가 정해져 있다. 예: 갑목은 해에서 장생, 자에서 목욕, ..., 묘에서 제왕. 병화는 인에서 장생, ..., 오에서 제왕.

표 도출 원리: 양간은 자기 본기 지지에서 제왕. 음간은 거꾸로 (역행).

### 9.2 사주에 적용

일간 × 4지지 = 4개의 12운성 라벨.
- 시지/일지/월지/년지 각각의 단계
- 단계가 높을수록 (건록/제왕) 그 자리에서 일간 에너지 강함
- 낮은 단계 (절/묘/사) 자리에서 약함

이게 강약 점수의 보조 지표가 됨. 그리고 어느 자리에 어떤 단계냐로:
- 일지가 제왕 → 자기 자리에 정점 → 자존감/자기주장 강함
- 일지가 절 → 자기 자리에 바닥 → 자존감 흔들리기 쉬움

### 9.3 12운성 vs 십이신살 (별개)

이름 비슷한 십이신살(十二神煞)은 따로 있음. 12운성은 천간×지지 매트릭스, 십이신살은 년지/일지 삼합 + 12지지 자리 매핑. 헷갈리면 안 됨.

## 10. 납음(納音) — 60갑자가 내는 소리

60갑자 각 쌍에 별도 오행이 부여. 일반 오행과 다른 "숨은 본질".

배치 규칙: 60갑자 → 30개 납음 오행 (30쌍). 예:
- 갑자·을축 = 해중금 (바닷속의 금)
- 병인·정묘 = 노중화 (화로 속의 불)
- 무진·기사 = 대림목 (큰 숲의 나무)
...

각 납음마다 짧은 비유 (강·산·바늘·등불 등)와 풀이가 따라옴. 일주의 납음이 가장 중요. 정통 명리학에선 보조 도구. 현대에선 콘텐츠 생산용.

## 11. 대운(大運) — 10년 단위 운의 흐름

### 11.1 시작 나이 계산

```
1. 절기 기준으로 년주 천간의 음양과 성별로 순행/역행 결정:
   - 양남(년간 양 + 남자) → 순행
   - 음녀(년간 음 + 여자) → 순행
   - 음남(년간 음 + 남자) → 역행
   - 양녀(년간 양 + 여자) → 역행

2. 출생일에서 다음 절기(순행) 또는 이전 절기(역행)까지의 일수 측정.
   12절(節)만 사용: 입춘·경칩·청명·입하·망종·소서·입추·백로·한로·입동·대설·소한.

3. 일수 ÷ 3 = 대운수 (시작 나이).
   (3일 = 1년의 명리학적 환산. 1년 = 1바퀴 ≈ 365일이고 1달 절기 ≈ 30일이라
    절기 30일을 10년으로 환산하면 3일=1년)

4. 나머지 처리: 1이면 버림, 2면 +1.
```

### 11.2 대운 기둥

월주에서 시작:
- 순행: 월주 + 1, +2, +3, ... (60갑자 순서로)
- 역행: 월주 - 1, -2, -3, ...

각 대운: 시작 나이 ~ +9세 = 10년. 대운 8~10개로 평생 커버.

### 11.3 대운 풀이

각 대운의 천간/지지를 일간 기준 십성으로 라벨링 (대운 십성). 그 십성이 용신/희신이면 길운, 기신이면 흉운. 색깔로 표시 (파랑=길/빨강=흉)는 거기서 자동.

세운(매년)도 같은 룰. 월운(매월), 일운(매일), 시운(매시)까지 같은 알고리즘이 fractal로 적용됨.

## 12. 풀이 한 사이클 — 입력에서 출력까지 데이터 흐름

```
[생년월일시 + 성별]
        ↓
1. 만세력 변환 → 4기둥 8글자 (절기 보정 포함)
        ↓
2. 지장간 펼치기 → 각 지지의 본기/중기/여기 확인 → 8글자가 사실은 ~16글자
        ↓
3. 십성 라벨링 → 일간 기준 7글자 + 지장간 모두에 십성 부여
        ↓
4. 5카테고리 카운트 (비겁/식상/재성/관성/인성)
        ↓
5. 강약 판정 → 득령·득지·득시·득세 + 점수화 (§3)
        ↓
   ┌─────┴─────┐
   ↓           ↓
6a. 격국       6b. 용신 (3학파 분기)
    (월지 본기      ├ 억부: 강약 기반
    투간 룰)        ├ 조후: 한난조습 기반
                   └ 격국: 격이 살아나는 흐름
        ↓
7. 희신·한신·기신·구신 라벨링 (용신 결정 후 자동)
        ↓

[모든 fact가 컴파일된 풀이 컨텍스트]
```

이 컨텍스트가 다 모이면 — 상담사(인간이든 LLM이든)가 prose로 변환만 하면 됨.

각 단계는 **결정적**이다. 같은 입력 → 같은 출력. 명리학 풀이의 "예술" 부분은 이 13단계가 아니라 **14단계 이후의 prose 작문**. 13단계는 100% 알고리즘.

---

## 13. 사이트마다 풀이가 달라질 수 있는 지점들

이건 **명리학 책에 적힌 일반론**이지, 특정 사이트가 어디서 분기하는지 확인한 게 아니다. 직접 비교 캡처는 thegot 1개만 했음.

같은 사주를 넣어도 사이트마다 풀이가 달라질 수 있는 지점:

1. **만세력 자체 정확도** — 절기 timestamp의 정밀도
2. **진태양시 / 서머타임 / 야자시 보정** — on/off 토글 차이
3. **강약 점수 가중치** — 학파마다 월령:일지:시지:천간:지장간 가중치가 다름
4. **용신 학파 우선순위** — 억부 우선이냐, 조후 우선이냐, 격국 우선이냐
5. **변격 임계값** — 종강격으로 보내는 기준점
6. **신살 데이터셋 크기** — 100가지 신살 중 몇 개를 평가하느냐
7. **prose의 톤·길이** — 같은 fact 컨텍스트에서도 어떻게 풀어쓰느냐

---

## 14. 결론

사주 풀이는 명리학 책 기준으로는 **결정적 알고리즘 + 그 위에 얹는 자연어 prose** 구조. 알고리즘 부분은 자평진전·적천수·궁통보감에 적힌 절차고, 현대 사이트들이 그걸 자동화한 거라고 추정되지만 — **각 사이트의 코드를 본 적은 없다**. 출력만 보고 역추정한 결론임.

직접 짠다면 핵심 모듈 우선순위:

| 순위 | 모듈 | 이유 |
|---|---|---|
| 1 | 지장간 펼치기 + 통근/투간 판정 | 모든 후속 단계의 기반 |
| 2 | 5카테고리 카운트 + 강약 점수 | 풀이의 첫 분기점 |
| 3 | 용신 (억부 1종 우선) | 모든 권고의 출처 |
| 4 | 격국 (정격 12개) | 사주의 구조 라벨 |
| 5 | 십성 조합 패턴 (관인상생 등) | 풀이의 깊이 |
| 6 | 합/충/형 트리거 | 이미 우리 코드에 있음 |
| 7 | 신살 (15-20종) | 풀이 풍성도 |
| 8 | 대운 (이미 있음) + 길흉 자동 판정 | 시간축 |
| 9 | 12운성 (보조) | 보조 신호 |
| 10 | 납음 (보조) | 콘텐츠 생산용 |

1-5번까지 짜면 사주 풀이의 80%가 자동화됨. 나머지는 prose의 영역.
